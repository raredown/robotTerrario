Formato del fichero

El fichero contenedor de las solicitudes de portabilidad, ha de tener el siguiente formato para que todas y cada una de las solicitudes que en él se encuentran puedan ser cargadas en la base de datos de manera satisfactoria.

 

Cada solicitud completa ha de tener como último caracter un salto de carro.
Cada campo dentro de una solicitud ha de estar separado por un "punto y coma" (;).
Como máximo puede haber 70 rangos de numeración(NRN + Número Inicial + Número Final).
Cada campo tiene una longitud máxima y , esta, no puede ser excedida en ningun caso.
