Consulta Estado Número

Esta consulta nos permite conocer el estado de un número que se encuentre en la base de datos de Portanode. Para que el número se encuentre en la base de datos debe haber pertenecido a algún proceso. Si el operador de Portanode que realiza la consulta no estaba implicado en el proceso, la información del número se obtiene por la adquisición.

Exporta la información a un fichero para poder abrirlo con Excel
