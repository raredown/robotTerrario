Cancelación del proceso de baja

Este mensaje le permite cancelar procesos de baja (mensajes NB) ya iniciados

Lo primero que se debe de hacer para proceder a una cancelación de baja es localizar el proceso que deseamos cancelar, para esto el sistema nos presenta dos posibilidades de búsqueda.

Por identificador de proceso:
Por Rango:

Usando cualquiera de las dos formas y tras pulsar el botón "Buscar", Nos aparecerá el proceso de baja buscado.

Activando el botón de Radio asociado a cada línea, seleccionaremos el proceso de baja que deseamos cancelar. Al pulsar el botón Enviar se iniciará la cancelación del proceso de baja (Mensaje XNB). La aplicación nos redijirá a una pantalla que indicará si el mensaje ha sido enviado de forma correcta o si se ha producido algún error.

Genera el mensaje de cancelación del proceso de baja seleccionado y lo envia.
