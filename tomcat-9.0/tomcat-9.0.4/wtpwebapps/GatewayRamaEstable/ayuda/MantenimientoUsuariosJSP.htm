Usuarios de la Aplicación

Desde este programa podremos realizar todas las operaciones de administración de usuarios.

El Sistema nos visualizará un relación de todos los usuarios.

Las opciones de que disponemos en este punto son:

    1. Crear nuevos usuarios.- para lo que pulsaremos el botón “Alta”.
    2. Modificar un usuario.- activaremos el radio asociado al usuario a modificar y pulsaremos el botón “modificación”.
    3. Eliminar un usuario.- seleccionaremos el usuario como en el caso anterior,
