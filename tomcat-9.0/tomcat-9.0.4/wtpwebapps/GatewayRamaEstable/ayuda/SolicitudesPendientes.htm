Solicitud de Cambio de Operador (por Fichero)

Las solicitudes deberán de estar previamente cumplimentada y catalogada en un archivo en disco.

Este proceso se compone básicamentede dos pasos:

  • El primero, es la carga de fichero con la información de los abonados a cambiar de Operador.
  • El segundo, consiste en el envió de mensajes a la Entidad de Referencia.

Para el primer paso, deberá escribir el nombre del fichero o pulsar el botón "Examinar"

para el segundo paso deberá de pulsar sobre el literal "Ver solicitudes pendientes".

Cargar Datos .- Las solicitudes deberán de estar previamente cumplimentada y catalogada en un archivo en disco. para activar el proceso se debe de escribir el PATH del archivo en la caja correspondiente “Cargar Datos del Fichero” o pulsar el botón "Examinar" y abrir la una ventana donde podrá navegar por la estructura lógica de los directorios del disco, seleccionando el archivo adecuado, esta operación rellenará la caja de texto con el nombre y Path del fichero.

Una vez tengamos el nombre del fichero, pulsaremos el botón “Cargar”. Esta acción desencadena un proceso de lectura y validación de la información que contiene el fichero. Si se detecta alguna inconsistencia o violación de las reglas de configuración del fichero, el sistema mostrará una ventana con las anomalías detectadas. En caso contrario, la información quedará en situación de ser transferida.

NORMAS PARA LA CREACIÓN DEL ARCHIVO DE TRASPASO

  • Cada Solicitud completa ha de tener como último carácter un salto de carro.
  • Cada campo irá separado del siguiente por el carácter “;”.
  • Como máximo puede haber 70 rangos de numeraciones (NRN + Número Inicial + Número Final).
  • La longitud máxima de los campos según la tabla adjunta, no podrá ser excedida bajo ningún concepto.
